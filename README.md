# Installation 
Checkout the source code from SVN
Ensure Anypoint Studio has devkit components installed
Import the connector project into studio workspace
Right-Click and select Anypoint Connector -> Install or Update

#Usage
After installing, search for 'smb' in the mule palette.
Drag and drop in the flow.
Create a new connection if the connector is declared for the first time and provide all the properties
